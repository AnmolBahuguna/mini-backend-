# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release of Mini Backend
- Threat detection API with multiple detection services
- User authentication with JWT
- Alert system with real-time notifications
- Evidence vault for storing security evidence
- Dashboard with threat analytics
- Community reporting features
- Women safety module
- API rate limiting and throttling
- Celery task queue integration

### Changed
- Improved API response times
- Enhanced error handling

### Fixed
- Fixed JWT token expiration issues
- Corrected CORS configuration

## [0.9.0] - 2023-12-01

### Added
- Alpha release for testing
- Core threat detection features
- User registration and login
- Basic API endpoints

### Known Issues
- Performance optimization needed
- Documentation incomplete
