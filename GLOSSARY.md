# Glossary of Terms

## Threat Detection

**Threat Level**
- LOW: Minimal security risk
- MEDIUM: Notable risk requiring attention
- HIGH: Significant threat to be acted upon
- CRITICAL: Immediate action required

**False Positive**
- An alert triggered for benign content incorrectly classified as malicious

**Threat Intelligence**
- Information about security threats and actors used to inform security decisions

**Zero-Day**
- Vulnerability unknown to the vendor or public

## Authentication

**JWT (JSON Web Token)**
- Stateless authentication method using cryptographic tokens

**Access Token**
- Short-lived token used to authenticate API requests

**Refresh Token**
- Longer-lived token used to obtain a new access token

**Supabase**
- Backend-as-a-Service platform handling authentication and database

## Infrastructure

**Celery**
- Asynchronous task queue for background job processing

**Redis**
- In-memory data store used for caching and Celery message broker

**PostgreSQL**
- Relational database management system

**Docker**
- Container technology for packaging applications

## APIs

**Rate Limiting**
- Restricting the number of requests per time period

**Throttling**
- Slowing down response when rate limits approached

**CORS**
- Cross-Origin Resource Sharing - controls cross-domain requests

**REST**
- Representational State Transfer - architectural style for APIs

## Security

**SSL/TLS**
- Encryption protocols for secure communication

**Bcrypt**
- Password hashing algorithm

**Hash**
- One-way cryptographic function producing fixed-length output

**Encryption**
- Converting data into unreadable form using keys

## Development

**Middleware**
- Software component processing requests/responses

**Serializer**
- Converting data between different formats (JSON, objects, etc.)

**Pagination**
- Dividing large result sets into manageable chunks

**Migration**
- Database schema change management
